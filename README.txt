Randall Ford Branding Website — GA4 Page Patch
================================================

Google Analytics measurement ID: G-EZGHJB5N5E

This package contains the six public HTML pages updated with the Google tag
immediately after the opening <head> element:

- index.html
- about.html
- tricon-residential.html
- cryptozoic-quidd.html
- cryptkins.html
- marvel-collect.html

Deployment
----------
Replace the matching HTML files in the website repository with these files.
This patch intentionally does not include or overwrite images, PDFs, CSS assets,
or other existing site files.

Verification
------------
After deployment, open the site and check Google Analytics Realtime. A visit may
take a short moment to appear. Browser privacy tools or ad blockers may prevent
the tag from firing during a test.
